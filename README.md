A Shell written in C for CS537 Introduction to Operating at UW Madison

Compile with make command

run shell with ./mysh

run a list of commands with ./mysh [filename]

close scipt with
```
exit
```
create alias with
```
alias [name of alias] [command]
```
run alias with
```
[name of alias]
```
remove alias with
```
unalias [name of alias]
```
